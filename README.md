## REACT based Google book search app

React-based Google Books Search app using useState and compenents. The app makes use of Node, Express and MongoDB to save books to review or purchase later. 

### Technologies Used
  1. JavaScript
  2. REACT
  3. Node.js
  4. Express
  5. MongoDB
  6. Bootstrap
